# Симулятор айтишника

Браузерная idle-игра на FastAPI: один игрок, состояние в памяти (без БД).

## Механики

### Статы

| Стата | Диапазон / тип | Старт |
|-------|----------------|-------|
| Энергия | 0–100 | 100 |
| Настроение | 0–100 | 100 |
| Деньги | целое число | 100 |
| Опыт (XP) | целое число | 0 |
| Уровень | целое число | 1 |
| Работа | строка | «офисный шнырь» |
| Ноутбук | строка или нет | нет |

Энергия и настроение после действий ограничиваются диапазоном 0–100.

### Действия

- **Работать** — энергия −15, настроение −5, деньги: случайная сумма в диапазоне дохода текущей работы, +10 XP. Недоступно, если энергия меньше 15.
- **Развлечения** (раздел внизу): **Прогулка** — 0₽, настроение +5; **Выпить пива** — 200₽, настроение +25; **Сходить в сауну** — 5000₽, настроение +70. Платные варианты недоступны, если не хватает денег.

### Уровни

Для перехода на следующий уровень нужно **N×100 XP**, где **N** — текущий уровень. При повышении уровня лишний XP переносится, уровень +1, энергия сбрасывается до 100.

### Карьера и ноутбуки

Должности отличаются **минимальным уровнем**, иногда **требуемым ноутбуком** и **диапазоном дохода** при работе. Устроиться можно через кнопки на главной, если выполнены уровень и ноутбук (если указан).

Ноутбуки покупаются за деньги; у игрока хранится один текущий ноутбук (покупка нового заменяет предыдущий).

### Лог

Внизу страницы — последние **5** сообщений о действиях и ошибках (недостаточно уровня, нужен ноутбук, не хватает денег и т.д.).

### Фоновая музыка

Файлы с расширением **`.mp3`** (регистр не важен: `.mp3`, `.MP3`) кладите в `audio/music/` — подхватываются по алфавиту и играются **по кругу**. Если один файл — он повторяется.

Браузеры часто блокируют автозапуск звука до клика: при необходимости появится кнопка «Включить музыку».
